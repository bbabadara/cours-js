Leçon 1/10

Introduction à JavaScript

JavaScript est un langage de programmation interprète, orienté objet, faiblement typé et principalement utilisé côté client (navigateur). Il permet de rendre les pages web dynamiques et interactives.

Objectifs

  • Comprendre ce qu'est JavaScript
  • Savoir où placer le code JavaScript
  • Écrire votre premier script

Où écrire JavaScript ?

1. Dans une balise script

<script>
  console.log("Hello World !");
</script>

2. Dans un fichier externe

<script src="script.js"></script>

3. Directement dans la console du navigateur

Appuyez sur F12 puis allez dans l'onglet Console

console.log()

C'est la fonction principale pour afficher des informations dans la console du navigateur. Très utile pour le débogage !

Leçon 2/10

Les Variables

Les variables sont des conteneurs qui permettent de stocker des données. En JavaScript, on utilise let, const et var pour les déclarer.

Différence entre let, const et var

  • let : portée de bloc, peut être modifié
  • const : portée de bloc, constante (ne peut pas être modifié)
  • var : portée fonctionnelle (ancienne méthode, à éviter)

Exemples

Déclarer avec let

let nom = "Alice";
let age = 25;
let estEtudiant = true;

console.log(nom);    // Alice
console.log(age);     // 25

Déclarer avec const

const PI = 3.14159;
const MESSAGE = "Bienvenue";

console.log(PI);       // 3.14159
console.log(MESSAGE);   // Bienvenue

Bonne pratique

Utilisez const par défaut, et let seulement si vous avez besoin de modifier la variable.

Leçon 3/10

Types de Données

JavaScript a plusieurs types de données fondamentaux. Le langage est faiblement typé, donc le type est déterminé automatiquement.

Les types principaux

String (Chaîne de caractères)

let prenom = "Alice";
let nom = 'Dupont';
let message = `Bonjour ${prenom} !`;  // Template literal

console.log(typeof prenom);  // string

Number (Nombre)

let entier = 42;
let decimal = 3.14;
let negatif = -10;

console.log(typeof entier);  // number

Boolean (Booléen)

let estActif = true;
let estMineur = false;

console.log(typeof estActif);  // boolean

Null et Undefined

let vide = null;       // Intentionnel
let indefini;          // Non défini

console.log(vide);     // null
console.log(indefini);  // undefined

Vérifier le type

Utilisez typeof pour connaître le type d'une variable.

Leçon 4/10

Conditions

Les conditions permettent d'exécuter du code en fonction d'un état. Elles utilisent if, else et switch.

if / else if / else

let age = 18;

if (age >= 18) {
    console.log("Majeur");
} else if (age >= 16) {
    console.log("Adolescent");
} else {
    console.log("Mineur");
}

Opérateurs de comparaison

== Égal (valeur)
=== Égal (valeur + type)
!= Différent (valeur)
!== Différent (valeur + type)
> Supérieur
< Inférieur
>= Supérieur ou égal
<= Inférieur ou égal

Switch

let jour = 1;

switch(jour) {
    case 1:
        console.log("Lundi");
        break;
    case 2:
        console.log("Mardi");
        break;
    default:
        console.log("Autre jour");
}
Leçon 5/10

Boucles

Les boucles permettent de répéter du code plusieurs fois. JavaScript offre plusieurs types de boucles.

for

Idéale quand on connaît le nombre d'itérations.

// for (initialisation; condition; incrémentation)
for (let i = 0; i < 5; i++) {
    console.log("Itération " + i);
}

while

Répète tant que la condition est vraie.

let i = 0;
while (i < 3) {
    console.log("Compteur: " + i);
    i++;
}

do...while

Exécute AU MOINS une fois, puis teste la condition.

let i = 0;
do {
    console.log("Valeur: " + i);
    i++;
} while (i < 3);

for...of

Itère sur les éléments d'un tableau.

let fruits = ["pomme", "banane", "orange"];

for (let fruit of fruits) {
    console.log(fruit);
}
Leçon 6/10

Fonctions

Les fonctions sont des blocs de code réutilisables. Elles peuvent être declarées de plusieurs façons.

Fonction nommé

function saluer(nom) {
    return "Bonjour " + nom + " !";
}

let message = saluer("Alice");
console.log(message);

Fonction anonyme

Stockée dans une variable.

const direBonjour = function(nom) {
    return "Salut " + nom;
};

console.log(direBonjour("Bob"));

Fonction fléchée (Arrow Function)

Syntaxe courte, très utilisée en moderne JavaScript.

Avec plusieurs instructions

const addition = (a, b) => {
    let resultat = a + b;
    return resultat;
};

console.log(addition(5, 3));

Syntaxe courte (un seul paramètre)

const carre = x => x * x;

console.log(carre(4));  // 16

Paramètres par défaut

function dire(nom = "Inconnu") { ... }

Leçon 7/10

Tableaux

Les tableaux (Array) permettent de stocker plusieurs valeurs dans une seule variable.

Créer un tableau

let fruits = ["pomme", "banane", "orange"];

console.log(fruits[0]);   // pomme (index 0)
console.log(fruits.length); // 3

Méthodes utiles

push() - Ajouter à la fin

let couleurs = ["rouge", "vert"];
couleurs.push("bleu");
console.log(couleurs);

pop() - Retirer le dernier

let nombres = [1, 2, 3];
nombres.pop();
console.log(nombres);

forEach() - Parcourir

let animaux = ["chat", "chien"];

animaux.forEach(animal => {
    console.log(animal);
});

map() - Transformer

let nombres = [1, 2, 3];
let doublés = nombres.map(n => n * 2);

console.log(doublés);

filter() - Filtrer

let nombres = [1, 2, 3, 4, 5];
let pairs = nombres.filter(n => n % 2 === 0);

console.log(pairs);
Leçon 8/10

Objets

Les objets permettent de stocker des données sous forme de paires clé-valeur. C'est la base de JavaScript.

Créer un objet

let personne = {
    nom: "Alice",
    age: 25,
    ville: "Paris"
};

console.log(personne.nom);
console.log(personne["age"]);

Méthodes dans un objet

let personne = {
    nom: "Alice",
    age: 25,
    saluer: function() {
        return "Bonjour, je suis " + this.nom;
    }
};

console.log(personne.saluer());

Parcourir un objet

for...in

let personne = { nom: "Bob", age: 30 };

for (let cle in personne) {
    console.log(cle + ": " + personne[cle]);
}

Object.keys() / Object.values()

let personne = { nom: "Bob", age: 30 };

console.log(Object.keys(personne));
console.log(Object.values(personne));
Leçon 9/10

DOM (Document Object Model)

Le DOM permet de manipuler le contenu HTML avec JavaScript. Chaque élément HTML est un objet accessible.

Sélectionner des éléments

getElementById

// <div id="titre">
// </div>

document.getElementById("titre");

querySelector

// Premier élément .classe
document.querySelector(".maClasse");

// Premier <p>
document.querySelector("p");

querySelectorAll

// Tous les éléments .classe
document.querySelectorAll(".maClasse");

Manipuler le contenu

innerHTML vs textContent

element.innerHTML = "<b>Gras</b>"; // HTML interprété
element.textContent = "Texte brut"; // Pas d'interprétation

Modifier les attributs

element.getAttribute("src");
element.setAttribute("alt", "Image");

Modifier le style CSS

element.style.color = "blue";
element.style.backgroundColor = "#fff";

Créer des éléments

// Créer un élément
let div = document.createElement("div");

// Ajouter du contenu
div.textContent = "Nouvelle div";

// L'ajouter au parent
document.body.appendChild(div);
Leçon 10/10

Événements

Les événements permettent de réagir aux actions de l'utilisateur (clic, saisie, etc.)

Types d'événements

Souris

click, dblclick, mouseover, mouseout

Clavier

keydown, keyup

Formulaire

submit, focus, blur

Fenêtre

load, resize, scroll

Méthode 1: onEvent

element.onclick = function() {
    console.log("Clic détecté !");
};

Méthode 2: addEventListener

Méthode recommandée, permet plusieurs écouteurs.

element.addEventListener("click", function() {
    console.log("Clic !");
});

element.addEventListener("click", () => {
    console.log("Autre écouteur");
});

Objet Event

element.addEventListener("click", function(e) {
    console.log(e.target);    // Élément cliqué
    console.log(e.type);     // "click"
    console.log(e.clientX);  // Coordinnée X
    console.log(e.clientY);  // Coordinnée Y
});
